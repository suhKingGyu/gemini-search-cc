# Security Policy

## Supported Versions

Currently, only the latest release of the project is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please **do not** report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability, please report it via email to the maintainer. We will acknowledge receipt of your vulnerability report as soon as possible and strive to send you regular updates about our progress.

If you have not received a reply to your email within 48 hours, or have not heard from the security team for the past 5 days, please follow up with another email.
